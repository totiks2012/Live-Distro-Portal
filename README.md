# Live-Distro-Portal
A simple local portal for live Linux distributions. Displays utilities and help in a single browser window.
# LiveDistro Portal
## Опробовано на live дистрибутиве Minios https://minios.dev/

Простой локальный портал для live-дистрибутивов Linux. Показывает утилиты и справку в одном окне браузера.

## Для чего

В live-системах часто есть специфичные утилиты (менеджеры сессий, ядер, установки), но они спрятаны в меню или недокументированы. Этот инструмент делает их видимыми и доступными через единый интерфейс.

## Как работает

- Читает конфигурационный файл `ldp.conf` — в нём перечислены утилиты: имя, команда запуска, путь к иконке.
- Отображает Markdown-файлы из папки `help_doc/` как отформатированную справку.
- Использует фон `wal.png` из папки `Wal/` (если есть).
- Запускает приложения с полным пользовательским окружением (работает даже с GTK-приложениями).

## Конфигурация

Создайте `ldp.conf` рядом со скриптом:

```text
***
Minios Configurator
pkexec /usr/bin/minios-configurator
/usr/share/icons/Adwaita/48x48/categories/preferences-system.png
***
Minios Session Manager
/usr/bin/minios-session-manager
/usr/share/icons/Adwaita/48x48/devices/media-floppy.png
***
```

Каждый блок между `***` — это одна утилита:
1. Имя (отображается на ярлыке)
2. Команда запуска
3. Путь к PNG-иконке

## Структура проекта

```
LDP/
├── main.py
├── ldp.conf          ← ваш конфиг
├── help_doc/         ← *.md файлы
└── Wal/wal.png       ← фон (опционально)
```

## Запуск

```bash
./run-portal.sh
```

Скрипт автоматически:
- проверит наличие Python и pip,
- создаст виртуальное окружение,
- установит зависимости (`eel`, `markdown`),
- запустит портал в браузере.

## Что решено

- **Фон всегда подгружается** — используется HTTP-роут вместо `file://`.
- **Иконки отображаются корректно** — копируются в `web/icons/` и доступны по HTTP.
- **Приложения запускаются как из терминала** — с `DISPLAY`, `PATH`, `gi` и всем окружением.
- **Нет гонок и задержек** — всё работает мгновенно и стабильно.

## Требования

- Python 3
- Возможность записи в домашний каталог (для виртуального окружения)

---

Инструмент не претендует на универсальность — он решает одну задачу: сделать специфичные возможности live-дистрибутива легко доступными. Получить доступ к специфичным утилитам скриптам, и справке из единого центра запуска. Ярлык на эту утилиту можно поместить на рабочий стол.
